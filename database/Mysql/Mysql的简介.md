# Mysql的简介

## Mysql优点

- Mysql数据库隶属于Mysql AB公司，总部位于瑞典，后来被Oracle收购
- 优点：
  - 成本低：开放源代码，一般可以免费试用
  - 性能高：执行很快
  - 简单：很容易安装和使用

## DBMS的分类

- 基于共享文件系统的DBMS
- 基于客户机-服务器的DBMS(MYSQL、ORACLE、SqlServer)
